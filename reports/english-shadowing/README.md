# 英文影子跟读

## 定位

一份面向成年人的每日英文影子跟读：语言难度保持友好，选材必须成熟、有现实感，并且可以真正开口练习。

内容主要来自名人演讲与深度访谈、成人经典书籍和成人向经典电影。优先选择真实人物面对选择、失败、责任、关系、自由、时间与重建的表达。每期只使用一句很短、经过核验的原文作为“声音锚点”，其余材料由编辑围绕主题原创或改写。

它不是工作英语日报，也不是单词教材。目标是通过经典内容训练听感、节奏、发音、表达和英语语感。

## 适合水平

- 主要难度：A2–B1
- 每期训练：约 12–15 分钟
- 语言：英文练习 + 中文讲解
- 发音参考：默认美式英语；明显的英美差异会单独标注

## 主要来源

1. 名人演讲、毕业致辞、获奖感言和访谈
2. 经典书籍中的短句与核心思想
3. 经典电影中的短台词和场景
4. 体育、文学、历史和文化人物的公开表达

## 每期包含

- 今日人物、作品或场景
- 一句极短原声及中文理解
- 背景故事
- 原创影子跟读短文
- 意群、重音、弱读和连读提示
- 5 个重点词语及 IPA 音标
- 四轮影子跟读训练
- 一个联系个人经历的开口问题
- 参考回答和可迁移句型
- 今日带走的一句

## 内容原则

- 有趣、有故事感，不写成考试教材。
- 原文必须核验，不能制造“名人名言”。
- 现代书籍、电影、演讲和访谈只引用极短片段。
- 不连续重复同一人物、作品或主题。
- 内容难度略有挑战，但整体必须大体听懂。
- A2–B1 只限制英文表达难度，不降低主题成熟度。
- 默认不选儿童文学、动画片、童话、青少年成长和童趣冒险题材。
- 选材气质以 2026-07-18 的 Steve Jobs 与 2026-07-21 的 J. K. Rowling 两期为正向标杆。
