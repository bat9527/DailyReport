# Web3 行情与生态日报

## 定位

面向长期持有 BTC、ETH 并希望理解 Web3 市场结构的每日行情与生态学习日报。

它不是实时交易终端，也不提供机械买卖信号。日报用固定口径记录价格、波动、杠杆、链上活动、重要项目进展和行业人物观点，帮助读者区分市场噪声、事实变化与利益相关者的叙事。

## 每日固定关注

- BTC、ETH 的美元价格、24 小时/7 天/30 天变化、成交与波动；
- ETH/BTC、BTC 市占率、整体市值与稳定币规模；
- BTC、ETH 永续合约的资金费率、未平仓量和现货/标记价格关系；
- Ethereum 及主要生态的 TVL、费用、DEX 活动和稳定币变化；
- 监管、ETF、协议升级、安全事件与宏观变量。

## 项目观察池

观察池不代表投资推荐，也不要求每天全部出现。只有出现可核验、足以改变理解的新进展时才写入。

- 主流网络：Bitcoin、Ethereum、Solana、BNB Chain
- 扩容网络：Arbitrum、Optimism、Base
- DeFi：Uniswap、Aave、Sky、Lido
- 基础设施：Chainlink、EigenLayer
- 稳定币：Tether、Circle / USDC

## 人物观点池

重点跟踪 Vitalik Buterin、Brian Armstrong、Michael Saylor、Arthur Hayes、Anatoly Yakovenko、Changpeng Zhao、Hayden Adams、Stani Kulechov 等人的公开观点。

人物在观察池中不代表其观点可靠。每条观点必须提供原始出处、发布日期、上下文和利益关系，并附上事实边界或反方证据。没有高质量新观点时明确省略，不用旧言论凑数。

## 隐私与安全

公开仓库中只记录资产观察列表，不保存持仓数量、成本价、钱包地址、交易所账户、API 密钥或私钥。日报不得连接钱包、读取个人账户或执行交易。

## 阅读方式

- 对话版：约 5 分钟，先看市场状态、异常指标和三件大事。
- GitHub 版：约 12–15 分钟，保留完整数据时间戳、项目进展、人物观点、风险雷达和来源。
