# 金融科技出海日报

## 定位

面向个人能力成长和行业视野拓展的金融科技学习日报。

以非洲为核心，持续理解金融科技、数字银行、支付、信贷、风险管理和中国企业出海。它不是工作周报，不追踪个人项目进度，也不强行给出任务清单。

## 覆盖权重

- 非洲金融科技：约 60%
- 中国企业及金融科技能力出海：约 20%
- 全球标杆案例与前沿趋势：约 20%

权重是筛选方向，不要求每天机械凑齐。

## 核心主题

1. 非洲金融监管与市场变化
2. 数字银行、移动钱包和支付网络
3. 数字信贷、普惠金融与商业模式
4. 风控、反欺诈、KYC、征信和催收
5. 银行核心、开放银行、云和 AI 应用
6. 中国公司、技术和模式的海外落地
7. 值得研究的公司、产品、失败案例和数据

## 阅读方式

- 对话推送：约 5 分钟读完，聚焦最重要的内容。
- GitHub 完整版：约 15–20 分钟读完，包含背景、机制、案例和完整来源。
- latest.md：始终指向或保存最新一期。
- archive：按照北京时间日期长期归档。

## 不包含

- 个人项目进度
- 日常工作任务提醒
- 没有实质信息的公司宣传稿
- 缺乏来源的市场传闻
- 为了数量而拼凑的普通新闻
